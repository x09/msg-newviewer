# MSG NewViewer

Лёгкий просмотрщик сохранённых файлов электронной почты Microsoft Outlook
(`.msg`) для Linux. Не требует установленного Microsoft Outlook и вообще
никаких проприетарных или сторонних зависимостей — только Python 3 и Tkinter
из стандартной поставки.

Приложение самостоятельно разбирает бинарный формат `.msg` (OLE-хранилище),
показывает заголовки, тело письма, список получателей и вложения, а также
умеет сохранять вложения на диск и конвертировать письма в `txt`/`html` из
командной строки.

## Возможности

- Просмотр заголовков письма: тема, отправитель, получатели (To/Cc/Bcc), дата.
- Отображение тела письма в текстовом и HTML-представлении.
- Список вложений с возможностью сохранения любого из них в исходном формате.
- Поддержка вложенных писем (`.msg` внутри `.msg`) и встроенных объектов.
- Графический интерфейс на Tkinter и неинтерактивный режим командной строки.
- Корректная работа с кодировками и Unicode-потоками MAPI.
- Ассоциация с типом `application/vnd.ms-outlook` — открытие `.msg` двойным
  щелчком в файловом менеджере.

## Технические основы

Реализация опирается исключительно на открытые спецификации Microsoft:

- **[MS-CFB]** — Compound File Binary File Format: OLE-хранилище, заголовок,
  сектора FAT/DIFAT/MiniFAT, дерево каталога.
- **[MS-OXMSG]** — Outlook Item (.msg) File Format: потоки
  `__substg1.0_XXXXTTTT`, `__properties_version1.0`, хранилища получателей
  `__recip_version1.0_#N`, вложений `__attach_version1.0_#N` и вложенных
  писем (`0x3701000D`).
- **[MS-OXPROPS]** — каталог тегов и типов данных MAPI.

Весь разбор формата написан с нуля на чистом Python и не использует внешних
библиотек.

## Требования

- Python 3.6 или новее.
- Tkinter (пакет `python3-modules-tkinter` в ALT Linux,
  `python3-tk` в Debian/Ubuntu) — нужен только для графического режима.
  Режим `--convert` работает и без Tkinter.

## Установка

### Из RPM-пакета (ALT Linux)

```bash
rpm -i msg-newviewer-1.0-alt1.noarch.rpm
```
или , если установлен репозиторий altrepo.ru, штатным способом

```bash
apt-get install msg-newviewer
```


При установке пакет регистрирует MIME-тип `application/vnd.ms-outlook`,
устанавливает `.desktop`-файл и иконки, а также прописывает приложение
обработчиком `.msg` по умолчанию в `/etc/xdg/mimeapps.list`.

### Из исходников (без установки)

```bash
python3 msg-newviewer            # графический интерфейс
```

Файл самодостаточен — достаточно сделать его исполняемым:

```bash
chmod +x msg-newviewer
./msg-newviewer
```

## Использование

### Графический интерфейс

```bash
msg-newviewer               # открыть пустое окно
msg-newviewer письмо.msg    # сразу открыть указанный файл
```

Вложения сохраняются через интерфейс: выберите вложение в списке и укажите
каталог назначения.

### Командная строка (конвертация и извлечение)

Неинтерактивный режим сохраняет тело письма и все вложения в указанный
каталог, не открывая окно:

```bash
msg-newviewer --convert --mode=txt  --outdir=out письмо.msg
msg-newviewer --convert --mode=html --outdir=out *.msg
```

Параметры:

| Параметр           | Назначение                                             |
|--------------------|--------------------------------------------------------|
| `--convert`        | режим конвертации (графический интерфейс не открывается)|
| `--mode=txt\|html` | формат сохранения тела письма (по умолчанию `txt`)     |
| `--outdir=КАТАЛОГ` | каталог назначения (по умолчанию текущий)              |
| `-h`, `--help`     | краткая справка                                        |

Можно передавать сразу несколько `.msg`-файлов — каждый обрабатывается
отдельно, тело и вложения складываются в каталог `--outdir`.
